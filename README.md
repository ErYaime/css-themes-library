# 🔳 css-themes-library

> **Ecosistema de temas CSS modulares y ligeros, diseñados bajo una estética brutalista y técnica. Arquitectura escalable lista para usar.**

[![License: MIT](https://img.shields.io/badge/License-MIT-black.svg)](https://opensource.org/licenses/MIT)
![Version: 1.0](https://img.shields.io/badge/Version-1.0-blue.svg)
![Status: Active](https://img.shields.io/badge/Status-Active-brightgreen.svg)

Esta librería nace con el objetivo de proporcionar estilos sólidos, ángulos rectos y una jerarquía visual clara para aplicaciones que priorizan la eficiencia y el orden técnico.

---

## 💎 Temas Disponibles

### 01. Tema Prisma (v1.0)
El tema insignia de la librería. Inspirado en terminales técnicas y diseño industrial.
- **Estética**: Brutalista, ángulos de 0px, bordes marcados de 2px.
- **Características**: Header fijo con desenfoque gaussiano (Glassmorphism), menús laterales adaptativos y controles de rango personalizados con seguimiento de color.

---

## 🚀 Instalación Rápida (CDN)

No necesita compilación, preprocesadores ni dependencias complejas. Integre el **Tema Prisma** directamente desde GitHub usando **jsDelivr**.

Añada lo siguiente en el `<head>` de su documento HTML (reemplace `SuUsuario` con su nombre de perfil de GitHub):

```html
<link rel="stylesheet" href="[https://cdn.jsdelivr.net/gh/SuUsuario/css-themes-library/01-tema-prisma/main.css](https://cdn.jsdelivr.net/gh/SuUsuario/css-themes-library/01-tema-prisma/main.css)">

<script src="[https://cdn.jsdelivr.net/gh/SuUsuario/css-themes-library/01-tema-prisma/prisma.js](https://cdn.jsdelivr.net/gh/SuUsuario/css-themes-library/01-tema-prisma/prisma.js)" defer></script>
